# Wheel of Fortune — Bollywood Edition

An elegant Bollywood word-guessing game built with Next.js, React, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play.

## How to Play

1. **Spin the wheel** — Land on a value or hit Snake (bankrupt).
2. **Guess consonants** — Correct letters earn 2× the wheel value per occurrence.
3. **Buy vowels** — Costs 200 points; correct vowels earn 2× the wheel value per occurrence.
4. **Call Out Movie** — Guess the full title for +50,000 (or −50,000 if wrong).
5. **Complete 5 movies** — The team with the highest score wins the round.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
